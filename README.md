#  <p align="center">calorie-app</p>

Sample application using MEAN stack (MongoDB, Express.JS, Angular, Node.JS)

## Usage
Clone the project and run:
```
cd server
npm start
```

Open other terminal and run:
```
cd app
ng serve -o
```
